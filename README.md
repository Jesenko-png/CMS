# PHP CMS

A lightweight Content Management System (CMS) built with PHP for managing website content easily and efficiently.

## 🌐 Live Demo

Check out the live version of the CMS:  
👉 https://jesenko.free.nf/cms/?i=1

## 🚀 Features

- Create, edit, and delete pages/posts
- Admin dashboard
- User authentication (login/register)
- Email verification using PHPMailer
- File and image uploads
- Simple and clean UI
- SEO-friendly structure
